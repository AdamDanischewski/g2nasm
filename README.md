# m2nasm
Converts MASM to NASM 
```shell
Usage: m2nasm.bsh [-h] <file>

 <file>: Masm asm file.
         gcc -S -o code.masm code.c ## file=code.masm

 OPTIONS:
         -h|--help           Usage
```
