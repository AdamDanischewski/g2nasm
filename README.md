# g2nasm
Converts GAS ASM to NASM<br> 

```shell
Usage: g2nasm.bsh [-h] <file>

 <file>: Masm asm file.
         gcc -S -o code.gasm code.c ## file=code.gasm

 OPTIONS:
         -h|--help           Usage
```
